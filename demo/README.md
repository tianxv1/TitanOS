# TitanOS Demo

This folder contains demo materials for TitanOS.

## Structure

```
demo/
├── screenshots/   # Screenshots of TitanOS UI
├── videos/        # Demo videos
└── gif/          # Animated GIFs for quick preview
```

## How to Add Demos

### Screenshots
Add PNG/JPG screenshots of the TitanOS interface:
- Dashboard
- Timeline
- Knowledge Graph
- Goal Tree
- Multi-Agent Workflow
- Agent Marketplace

### Videos
Add MP4 demo videos showing:
- Creating a goal and watching AI auto-plan
- Multi-Agent collaborating on a task
- Growth dashboard updates
- Knowledge graph exploration

### GIFs
Create short animated GIFs (3-5 seconds) for:
- Quick feature overview
- Twitter/social media preview
- README preview
